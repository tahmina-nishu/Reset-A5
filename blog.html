Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a way for web pages to be represented in a structure that JavaScript can interact with. It turns the HTML of a page into a tree-like format, where each part of the page (like text, images, or buttons) is an object.
Using the DOM, you can change the content, style, or structure of a web page dynamically. For example, you can change the text of a heading or make a button appear or disappear, all while the page is open in a browser.

How do you select an element from the DOM?

To select an element from the DOM, you typically use JavaScript methods that help you identify and target specific elements on the page. Here are some common ways to select an element:

getElementById(): Selects an element by its id attribute.
getElementsByClassName(): Selects elements by their class name.
getElementsByTagName(); Selects elements by their tag name (like div, p, h1, etc.).
querySelector(): Selects the first element that matches a CSS selector.
querySelectorAll(): Selects all elements that match a CSS selector.

What is event delegation in the context of the DOM and why is it useful?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element instead of adding event listeners to individual child elements. When an event occurs on a child element, the event "bubbles up" to the parent, and the parent can handle the event for all its children.

It is useful for -
Efficiency: Instead of attaching many event listeners to each child element, you only need one listener on the parent.
Dynamic content: If new child elements are added later, the parent will automatically handle events for them too, without needing extra code.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using the DOM by accessing the element and then changing its properties.

1. Manipulating Attributes:
(a) To change an element's attribute (like id, src, href, etc.), you use the setAttribute() method.
(b) You can also get an attribute value using getAttribute()

2. Manipulating Styles:
(a) To change an element's style, you can directly modify the style property of the element.

For example:
let div = document.getElementById('myDiv');
div.style.backgroundColor = 'blue';
div.style.fontSize = '20px'; // Change font size
